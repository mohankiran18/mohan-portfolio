# Personal Portfolio Website v1

First version of my personal portfolio — built with plain HTML, CSS, and JavaScript before migrating to a React + Tailwind stack.

Live (current version): [mohan-kiran.netlify.app](https://mohan-kiran.netlify.app/)

---

## Context

This is the original portfolio built during my early frontend learning phase. It covers the same content — projects, skills, and contact — but implemented without any framework or build tooling. The current production portfolio has since been rebuilt in React with Tailwind CSS and deployed via Netlify.

Keeping this version public as a reference point for the progression from vanilla web development to component-based architecture.

---

| Category | Tools |

---

## What Changed in v2

| Feature | v1 (This Repo) | v2 (React + Tailwind) |
|---|---|---|
| Architecture | Plain HTML/CSS/JS | React component-based |
| Styling | Custom CSS | Tailwind CSS utility classes |
| Chatbot | None | AI chatbot via Chatbase |
| Contact Form | Static | EmailJS integration |
| Deployment | — | Netlify (CI/CD from GitHub) |
| Maintainability | Single file structure | Modular component structure |
